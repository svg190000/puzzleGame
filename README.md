# Jigsaw Puzzle Game

A beautiful React Native jigsaw puzzle game built with Expo.

## Features

- Upload images from gallery or camera
- Customizable difficulty (3x3 to 6x6 grid)
- Jigsaw-shaped puzzle pieces with connectors
- Drag and drop with smooth animations
- Piece snapping when near correct position
- Completion celebration with confetti
- Timer and move counter
- Save game progress locally

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

Then press `i` for iOS simulator, `a` for Android emulator, or scan QR code with Expo Go app.

## Tech Stack

- Expo
- React Native
- react-native-gesture-handler
- react-native-reanimated
- expo-image-picker
- AsyncStorage
